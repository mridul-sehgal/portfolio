# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

---

### 2️⃣ Navigate into the project

```bash
cd portfolio
```

---

### 3️⃣ Install dependencies

```bash
npm install
```

---

### 4️⃣ Run the development server

```bash
npm run dev
```

---

### 5️⃣ Open in browser

```
http://localhost:5173
```
